# CNN Metaheuristics

## Project Layout

- `src/`: core library code (datasets, model, training, algorithms, evaluation, plots).
- `scripts/`: runnable experiment entry points.
- `notebooks/`: interactive analysis and experimentation.
- `results/`: generated figures, tables, and model artifacts.
- `data/`: dataset cache and data files.

## Run Scripts

From the project root:

```bash
python scripts/train_baseline.py
python scripts/run_random_search.py
python scripts/run_compare_baselines.py
python scripts/run_all_methods.py
```
